Este projeto tem como intuito implementar um jogo da velha em Java, com jogabilidade a partir do terminal.

Alguns pontos a serem melhorados:
- Jogo precisa estar em loop e usuário descide quando finalziar. (se der empate, continua) - Done
- Algumas jogadas (diagonal) não são reconhecidas como vitória.
- Quando usuário faz uma jogada que já foi realizada, jogada é invalidada e passar para o próximo jogador.
- Controle de erro caso jogador faça uma jogada fora dos limites permitidos. - Done
